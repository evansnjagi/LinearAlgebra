# House price prediction

We want to predict house prices based on two features: square frontage and number of rooms.

## Dataset

| id | Square feet | Rooms | Price (USD)|
|---|---|---|
|1 | 1000 | 2 | 200 |
|2 | 2000 | 4 | 400 |

## Task

1. Look at the features. Is house 2 a scaled version of house 1?
2. What would you get if you put these features into a matrix and find RREF?
3. Can these matrix be inverted? Can we get a unique solution from this data.
